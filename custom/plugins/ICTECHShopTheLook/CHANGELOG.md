# Changelog

## 1.0.0
- Initial release of ICTECHShopTheLook plugin
- Added "Shop The Look" CMS element with hotspot product linking
- Added "Shop Look Slider" CMS element for image sliders
- Variant selection support with add-to-cart functionality
- "Add All to Cart" button for adding all look products at once
- Product detail page extension showing related "Shop The Look" products
- Configurable layout styles: Image → Products, Products → Image, Only Image, Only Products
- Support for en-GB and de-DE storefronts
